### Статические методы
Поскольку Promise - это класс, то как и любого другого класса у него могут быть
статические методы.

Например:  
### Resolve
```Promise.resolve()```

Все промисы распакуются и в then попадет 1.
```
Promise.resolve(
    Promise.resolve(
        Promise.resolve(
            1
        )
    )
).then(v => v + 1) // 2
```

```
const p = Promise.resolve(1);

Prmose.resolve(p) === p // true
```

### All
`Prmose.all`:  
Все что может быть итерируемым передается в туда, например, массив.
В массиве могут быть простые элементы или промисы.

Promise.all возвращает промис и резовлится в случае, если были зарезовлены все
внутренние промисы и реджектится, если был зареджекчен хотя бы один промис.

```
Promise.all([1, 2]).then(([a, b]) => a + b);
```

### allSettled
Принимает в себя iterable и возвращает все том же порядке, где аргументы не распакованы. И это
не совсем промисы, а объекты, которые хранят в себе статус и результат. В связи с этим он не
выбрасывает ошибки и всегда фулфилдится.

```
Promise.allSettled([1, 2]).then(([a, b]) => {
    if (a.status === 'fulfilled') {
        console.log(a.value); // Результат выполнения
    }
    
    if (a.status === 'rejected') {
        console.log(a.reason) // Сообщение об ошибке
    }
})
```

### Race
Принимает в себя iterable, например, массив, set...
Он резолвится или реджектится с тем, что пришло первым.

Условно, отправили 5 запросов на сервер и первый ответ будет нашим результатом.

Пример:
```
Promise.race([
    sleep(100).then(() => 10),
    sleep(200).then(() => 20),
])
    .then(console.log) // 10
```
Зачем это можно использовать?
Например, чтобы установить таймаут, по истечении которого должны прервать запрос.

```
Promise.race([
    doAction(),
    sleep(200).then(() => Promise.reject('Timeout'))
])
```

### Any
`Promise.any`:  
Очень похож на `Promise.race` с разницей в том, что он зафулфилдится в том случае,
если хотя бы один зафулфилдится и зареджектится только если все внутри зареджектятся.
